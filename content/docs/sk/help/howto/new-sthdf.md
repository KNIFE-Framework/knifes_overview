# HOWTO – Ako vytvoriť novú STHDF inštanciu

## 1. Predpoklady

- class template repo,
- pripravený Makefile,
- štruktúra `content/docs/sk/sthdf/`.

## 2. Vytvorenie novej triedy

1. Klonuj template.
2. Premenuj identifikátory (rok, študenti, projekty).
3. Spusti init.

## 3. Inicializácia

```bash
make init
make dev
```

## 4. Prepojenie na KNIFE

- A2 = Knowledge Contribution.
- študenti môžu tvoriť vlastné KNIFEs.

## 5. Overenie

- menu funguje,
- build bez chýb,
- obsah viditeľný.
