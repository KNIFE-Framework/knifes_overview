💡 **Čo je KNIFE?**  
Mikro-príspevok typu „mini-TEDx“:  
- 🎯 Čo rieši (účel, cieľ)  
- 🧩 Ako to rieši (princíp)  
- 🧪 Ako to použiť (aplikácia)  

Každý KNIFE má svoj vlastný folder a môže obsahovať text, obrázky, multimédiá alebo interaktívny obsah.
