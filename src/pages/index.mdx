## 🧾 Popis
**KNIFE** (Knowledge in Friendly Examples) je metodika na zdieľanie poznatkov  
v štruktúrovanej, prístupnej a znovupoužiteľnej forme.  

Tento web je **náhľadom** na prehľad a obsah aktuálne spravovaných KNIFE príspevkov.

---

## 📁 Štruktúra projektu
- `docs/` – dokumentácia
- `src/` – zdrojový kód (ak existuje)
- `tags.yaml` – metadáta
- `meta.json` – pre AI, API alebo Pages

## 🔗 Prepojenia

a je súčasťou vrstvy **CAS – Context Aware Solutions**.

## 🧪 Branch model – Dev/Prod

| Branch | Účel | Viditeľnosť |
|--------|------|--------------|
| `main` | Produkčné prostredie – hotový obsah. | Verejný |
| `dev`  | Vývoj – experimenty, úpravy. | Privátny |

---

## 🔄 Workflow

1. **Pracuj v `dev`** – nové stránky a úpravy.  
2. **Testuj lokálne** – napr. `npm run start`.  
3. **Merge `dev` → `main`** – cez PR alebo príkazy.  
4. **Publikuj z `main`** – GitHub Pages alebo iný hosting.
